Это итоговая работа по модулям вёрстки. Проект представляет собой страницу-рефлексию о пути обучения, выполненную с использованием современных технологий HTML и CSS.

Технологии, использованные в проекте:
- HTML5: семантическая разметка, использование тега <dialog> для модальных окон, работа со встроенными SVG-спрайтами (<symbol>, <use>).
- CSS3:
    - Flexbox для построения адаптивных макетов.
    - CSS-переменные: для управления цветами, шрифтами и отступами.
Функция clamp(): для реализации «резиновой» верстки без резких скачков.
Анимации и переходы (transition, @keyframes): сложная анимация иконки лайка и эффекты наведения на кнопках.
CSS-фильтры: индивидуальная стилизация каждой карточки.
Режимы смешивания (mix-blend-mode): для создания эффектов на вотермарках и кнопках.
- Адаптивность: страница корректно отображается на устройствах от 320px до десктопов.
Особенности реализации:
Шрифты: Использованы шрифты Inter (вариативный) и PressStart2P. Настройка веса выполнена через font-variation-settings.
- Фон: Реализован сложный фиксированный фон с использованием множественных градиентов (repeating-linear-gradient).
Иконки: В проекте используется SVG-спрайт для декоративных элементов и инлайновая SVG-анимация для сердца.
Модальное окно: Реализовано через нативный элемент <dialog> с применением псевдоэлемента ::backdrop.
Как запустить
Просто откройте файл index.html в вашем браузере.
